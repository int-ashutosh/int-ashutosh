## Projects

### Tweet Genie — Brand-Aware Tweet Writer · [Model](https://huggingface.co/intashutosh/suitegenie-tweet-writer) · *code available on request*
RAG-based promotional tweet generation system built for a startup. Multi-stage feature engineering pipeline using KeyBERT, spaCy NER, and BERT-based semantic expansion to annotate raw tweet data with style, goal, audience, niche, and brand context; search APIs used to populate real-time web context at scale. Fine-tuned a LoRA adapter on Qwen2.5-1.5B-Instruct via HuggingFace Transformers and TRL's SFTTrainer on a custom-cleaned dataset of ~1,100 rows.

`RAG` `LoRA` `PEFT` `Qwen2.5` `KeyBERT` `spaCy` `HuggingFace` `NLP`
